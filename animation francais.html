Introduction aux Transitions CSS

Les transitions CSS permettent de créer des animations fluides entre les différents états d'un élément. Il existe deux méthodes principales pour les implémenter.

Rappel : Les transitions nécessitent toujours un déclencheur comme :hover, :focus, ou des changements de classe via JavaScript.

Méthode 1: Syntaxe Raccourcie (Shorthand)

La méthode raccourcie permet de définir toutes les propriétés de transition en une seule ligne.

Syntaxe Raccourcie

.element { transition: all 0.5s ease; } .element:hover { background: #e74c3c; transform: scale(1.2); border-radius: 50%; }
Survolez-moi

Structure de la Syntaxe

transition: [property] [duration] [timing-function] [delay];
Exemples :
transition: all 0.3s;
transition: opacity 1s ease-in;
transition: transform 0.5s ease-out 0.2s;
transition: width 2s, height 1s;

Méthode 2: Propriétés Séparées

Vous pouvez également définir chaque propriété de transition individuellement.

Propriétés Séparées

.element { transition-property: color, margin-top, background-color; transition-duration: 1s, 5s, 2s; } .element:hover { color: yellow; margin-top: 30px; background: #9b59b6; }
Survolez-moi

Les Quatre Propriétés

transition-property

Définit quelles propriétés CSS seront animées

transition-property: color, transform, opacity;
transition-duration

Définit la durée de l'animation pour chaque propriété

transition-duration: 1s, 5s, 2s;
Important : L'ordre des durées dans transition-duration doit correspondre à l'ordre des propriétés dans transition-property.

Fonctions de Temporisation (Timing Functions)

Les fonctions de temporisation contrôlent la vitesse de l'animation durant sa durée.

ease (par défaut)

Démarrage lent, rapide au milieu, fin lent

Survolez

linear

Vitesse constante du début à la fin

Survolez

ease-in

Démarrage lent, fin rapide

Survolez

ease-out

Démarrage rapide, fin lente

Survolez

ease-in-out

Démarrage et fin lents, rapide au milieu

Survolez
Utilisation avec les propriétés séparées :
transition-timing-function: ease-in;
transition-timing-function: ease, linear, ease-in-out;

Synthèse : Les Quatre Propriétés Réduites à Une

Les quatre propriétés de transition peuvent être combinées en une seule déclaration dans cet ordre :

Ordre de la syntaxe raccourcie :
transition: [property] [duration] [timing-function] [delay];

Propriétés Séparées

.element { transition-property: transform, background-color; transition-duration: 1.5s, 2s; transition-timing-function: ease-in-out, linear; transition-delay: 0.5s, 0s; }

Syntaxe Raccourcie Équivalente

.element { transition: transform 1.5s ease-in-out 0.5s, background-color 2s linear; }
Survolez-moi
Règle importante : Dans la syntaxe raccourcie, l'ordre des valeurs doit toujours être :
property → duration → timing-function → delay

Introduction aux Animations CSS

Les animations CSS permettent de créer des effets visuels complexes en utilisant la règle @keyframes pour définir la séquence d'animation.

Structure de base :
@keyframes nomAnimation { ... }
element { animation: nomAnimation durée timing-function délai iteration-count direction; }

Animations de Base

Animations simples pour débuter avec les keyframes CSS.

Rotation

Rotation
@keyframes rotation { 0% { transform: rotate(0deg); } 100% { transform: rotate(360deg); } } .element { animation: rotation 3s linear infinite; }

Changement de Couleur

Couleurs
@keyframes changementCouleur { 0% { background-color: #e74c3c; } 25% { background-color: #3498db; } 50% { background-color: #2ecc71; } 75% { background-color: #f39c12; } 100% { background-color: #9b59b6; } } .element { animation: changementCouleur 5s infinite; }

Animation Complexe

Complexe
@keyframes animationComplexe { 0% { background-color: #e74c3c; transform: translateX(0) rotate(0deg); } 25% { background-color: #3498db; transform: translateX(50px) rotate(90deg); } 50% { background-color: #2ecc71; transform: translateX(100px) rotate(180deg); } 75% { background-color: #f39c12; transform: translateX(50px) rotate(270deg); } 100% { background-color: #9b59b6; transform: translateX(0) rotate(360deg); } } .element { animation: animationComplexe 6s infinite; }

Animation avec Délai

Délai 2s
.element { animation: couleurFluide 3s infinite 2s; }

Propriétés des Animations

Contrôlez le comportement de vos animations avec ces propriétés.

3 Itérations

3 fois
.element { animation: couleurFluide 2s 3; }

Direction Alternate

Alternate
.element { animation: couleurFluide 3s infinite alternate; }

Transitions vs Animations

Comprendre quand utiliser les transitions et quand utiliser les animations.

Transitions CSS

  • ✅ Simple à mettre en œuvre
  • ✅ Parfait pour les changements d'état simples
  • ✅ Animation automatique inverse
  • ❌ Pas de contrôle sur les états intermédiaires
  • ❌ Pas de boucle intégrée
  • ❌ Nécessite un déclencheur
Cas d'utilisation :
Boutons au survol
Menus déroulants
Changements de couleurs
Animations simples

Animations CSS

  • ✅ Contrôle total avec @keyframes
  • ✅ Boucles infinies possibles
  • ✅ Pas besoin de déclencheur
  • ✅ États intermédiaires personnalisables
  • ❌ Plus complexe à mettre en place
  • ❌ Pas d'animation inverse automatique
Cas d'utilisation :
Spinners de chargement
Animations complexes
Effets continus
Séquences d'animation
Recommandation : Utilisez les transitions pour les interactions utilisateur simples et les animations pour les effets visuels complexes et automatiques.

Terrain de Jeu CSS - Animations & Transitions

Exemples interactifs d'animations et de transitions CSS

Animations CSS

Créez des séquences complexes à plusieurs étapes en utilisant @keyframes.

Syntaxe d'Animation Description
@keyframes nom { 0% {} 100% {} } Définir la séquence d'animation
animation: nom 2s ease 1s 3 alternate Raccourci: nom, durée, temporisation, délai, itération, direction
animation-name: slideIn Spécifier quelles @keyframes utiliser
animation-duration: 2s Durée d'un cycle
animation-delay: 1s Délai avant le début
animation-iteration-count: 3 Nombre de répétitions (infinite pour infinie)

Changements de Couleur Instantanés

Utilisation de pourcentages qui se chevauchent pour des changements de couleur instantanés :

Couleurs Instantanées
Exemple de Code :
@keyframes couleurInstant {
  0%, 24% { background-color: red; }
  25%, 49% { background-color: blue; }
  50%, 74% { background-color: green; }
  75%, 99% { background-color: orange; }
  100% { background-color: purple; }
}

.element { animation: couleurInstant 5s infinite; }

Changements de Couleur Progressifs

Keyframes standard avec des transitions fluides entre les couleurs :

Couleurs Fluides
Exemple de Code :
@keyframes couleurFluide {
  0% { background: red; transform: scale(1); }
  25% { background: blue; transform: scale(1.1); }
  50% { background: green; transform: scale(1.2); }
  75% { background: orange; transform: scale(1.1); }
  100% { background: purple; transform: scale(1); }
}

.element { animation: couleurFluide 4s infinite; }

Transitions CSS

Créez des animations fluides entre les différents états d'un élément.

Syntaxe de Transition Description
transition: all 0.5s ease Raccourci: propriété, durée, temporisation, délai
transition: left 5s, color 7s Plusieurs propriétés avec la même temporisation
transition-property: color, transform Quelles propriétés animer
transition-duration: 1s, 2s Durée pour chaque propriété
transition-delay: 0s, 0.5s Délai pour chaque propriété
transition-timing-function: ease Courbe de vitesse de la transition

Transition Simple

Toutes les propriétés transitionnent avec la même temporisation :

Survolez-moi
Exemple de Code :
.element {
  transition: all 0.5s ease;
}

.element:hover {
  background: #e74c3c;
  transform: scale(1.2);
  border-radius: 50%;
}

Propriétés Multiples

Différentes propriétés avec des durées différentes :

Survolez-moi
Exemple de Code :
.element {
  transition: background 1s, transform 2s, border-radius 0.5s;
}

.element:hover {
  background: #2ecc71;
  transform: rotate(180deg);
  border-radius: 50%;
}

Transition avec Délai

La transition commence après un délai :

Survolez-moi
Exemple de Code :
.element {
  transition: all 1s ease 0.5s;
}

.element:hover {
  background: #f39c12;
  transform: translateX(50px);
}
Astuce : Cliquez sur n'importe quelle animation pour la mettre en pause/reprendre. Survolez les exemples de transition pour voir l'effet.
